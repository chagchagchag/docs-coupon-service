- redis enqueue

<br/>

`coupon-assigner`
- Redis 에 저장된 요청을 꺼내어서 MySQL에 저장하는 작업. 스케쥴링(@Scheduled) 기반의 데이터 배치 저장 처리
- 데이터 저장 작업 완료시 `UserAssignedNewCouponEvent` 라는 도메인 이벤트 발급(publish)
- @TransactionalEventListener 를 통해 트랜잭션 커밋이 완료된 couponId 를 Redis 내에 저장
